# inertia-boilerplate
> A strong starting point for creating applications using Laravel 12, Vue 3 and InertiaJS 2, with a ton of other convenience feature.

Below lists the included packages and features:
 - Laravel Pacakges
   - Laravel Debugbar
   - Spatie permission
   - Laravel Telescope
   - Pest Unit Testing
 - InertiaJS & Vue3
   - `@vueuse/core` and `@vueuse/component`
   - `chart.js` and `vue-chart3`
   - `unplugin-vue-components`
   - `unplugin-auto-import`
   - `element-plus`
   - `fullcalendar`
   - `vue3-openlays`
   - `tailwind.css`
   - `typescript`
   - `inertia`
   - `vitest`
   - `pinia`
   - `maska`
   - `dayjs`
